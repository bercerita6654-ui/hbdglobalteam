Tips Integrasi Google Sheets

  1. Klik tombol Template di atas untuk mengunduh format tabel standar.
  2. Buka Google Sheets baru, klik File > Impor > Upload file template tersebut.
  3. Setelah data diisi, klik File > Bagikan > Publikasikan ke web (Pilih: Seluruh Dokumen & .csv).
  4. Ganti link pada variabel SHEET_CSV_URL di dalam kode dengan link baru Anda agar otomatis tersinkronisasi.
